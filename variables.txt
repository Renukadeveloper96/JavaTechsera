Variable is a name of memory location.

three types of variables
1)local
2)instance
3)static


Local:a variable declared inside the body of the method is called local
you can use this variable only within that method and other methods in the class
local variable cannot be defined with "static" keyword

instance:
a variable declared inside the class but outside the body of the method is called
instance variable
it is not declared as static

static:
a variable that is declared as static is called a static variable
it cannot be local
